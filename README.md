## Setup
To run this project, install locally:
```
$ Python #.x Install
$ pip install
$ pandas install
$ numpy install
$ matplotlib install

Clone the repository:
$ git https://github.com/Nstrzelczyk/Kickstarter_Practice_Data_Science
```
